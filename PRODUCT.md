# Construction Budget Agent — ИИ-агент девелоперской финансовой модели

Веб-сервис, который за минуту превращает ключевые параметры участка и объекта в полную финансовую модель девелоперского проекта — и честно говорит, стоит ли в него входить.

## Для кого

- **Девелопер** — быстрая проверка участка и конфигурации проекта до сделки.
- **Инвестор** — независимая оценка экономики и рисков перед вложением.
- **Банк / кредитный аналитик** — экспресс-проверка проекта по критериям проектного финансирования.

## Главный принцип

Агент не подгоняет расчёты под желаемый результат. Если участок переоценён — он напишет «покупка участка экономически нецелесообразна». Если проект не пройдёт банк — «проект НЕ пройдёт банковское проектное финансирование» с перечнем причин. Все допущения помечаются явно, каждый расчёт имеет trace с формулами.

## Что считает

| Блок | Содержание |
| --- | --- |
| Бюджет | Детальный бюджет по главам и статьям, СМР с разбивкой на материалы/работы/механизмы |
| Цена и выручка | Рекомендованная цена продажи, сверка с рыночной, план продаж |
| **Оценка земли** | Остаточный метод: максимально обоснованная цена участка, безубыточная цена, жёсткий вердикт с защитой от завышенной цены продажи |
| **Банк (эскроу, 214-ФЗ)** | Эскроу-модель проектного финансирования: blended-ставка по покрытию, капитализация процентов, раскрытие на вводе, LLCR, 8 банковских критериев, стресс-тесты −10% цена / +10% себестоимость, вердикт approved/conditional/rejected |
| **Техприсоединение (ТУ)** | Нагрузки (электро, вода, канализация, тепло, газ) по нормативам, плата за ТП, сверка с бюджетом, контроль сроков мероприятий |
| Финансы | ГПР (S-кривая), кредитный график, ДДС, DSCR помесячно |
| Аналитика | Риски, сценарии (база/оптимизм/стресс), оптимизация до рыночной цены, план улучшений |
| **ИИ-заключение и чат** | Аналитическая записка для инвестора и кредитного комитета (OpenAI) + чат по проекту в свободной форме — ответы только по данным модели, без приукрашивания |
| Excel | Модель на 19 листах: от вводных до банковского блока и ТУ |

## Технологии

React + TypeScript (кабинет), FastAPI (API с авторизацией, квотами и rate-limit), Streamlit (внутренний кабинет), Docker / Render (деплой), OpenAI API (ИИ-заключение и чат), ЮKassa (оплата). 126 автотестов. Чек-лист вывода в сеть — `DEPLOY_CHECKLIST.md`.

## Развёртывание

- **Облако**: Render по `render.yaml` (один сервис: nginx + FastAPI + собранный React). Подробно — `README_RENDER.md`.
- **Свой сервер**: `docker-compose up` (backend + frontend). Подробно — `README_DEPLOY.md`.
- **Локально**: Streamlit-кабинет или uvicorn + Vite. Подробно — `README.md`.

Ключевые переменные окружения: `AUTH_SECRET`, `OPENAI_API_KEY` (опционально), `ALLOW_PUBLIC_GENERATE=false` для прода, `CORS_ORIGINS`, `RATE_LIMIT_*`.

## Безопасность и доступ

- Авторизация: PBKDF2-хэши паролей, Bearer-токены, роли admin/user.
- Публичные legacy-endpoints отключаются флагом для онлайн-доступа.
- Лимиты на тяжёлые операции: расчёты модели и ИИ-заключения (защита от расхода CPU и токенов).
- Секреты только через переменные окружения.

## Черновик тарифов (для обсуждения)

| Тариф | Кому | Включено | Ориентир цены |
| --- | --- | --- | --- |
| Старт | Частный девелопер | 1 пользователь, 30 расчётов/мес, Excel | 9 900 ₽/мес |
| Команда | Девелоперская компания | 5 пользователей, безлимит расчётов, ИИ-заключения, история проектов | 29 900 ₽/мес |
| Корпоративный | Банк / фонд | Свой сервер или выделенный инстанс, свои пороги критериев, white label | от 150 000 ₽/мес |

Цены — заготовка для проверки спроса, не оферта.

**Как работает оплата.** Тарифы с месячными квотами (расчёты модели и ИИ-вызовы) зашиты в биллинг-сервис; квоты проверяются на каждом тяжёлом запросе (HTTP 402 при исчерпании). Оплата картой — через ЮKassa: кнопка в разделе «Тариф» создаёт платёж и редиректит на страницу оплаты, вебхук `payment.succeeded` перепроверяется запросом к API ЮKassa и продлевает подписку на 30 дней. Если ключи ЮKassa не заданы — автоматический режим «оплата по счёту»: админ активирует тариф командой `POST /api/admin/users/{login}/plan`. Новые пользователи получают триал (5 расчётов и 5 ИИ-вызовов в месяц); просроченная подписка деградирует до триал-квот, данные не блокируются.

## Дорожная карта (кандидаты)

- Разбор загруженных документов (ТУ, сметы подрядчиков) с сверкой против модели
- Самостоятельная регистрация пользователей
- Региональные справочники ставок ТП и цен продаж
- Сравнение нескольких участков в одном расчёте

## Ограничения (честно)

- Нормативные ставки (себестоимость, плата за ТП, рыночные цены) — усреднённые допущения; для сделки обязательна проверка по фактическим ТУ, сметам и аналитике рынка.
- ИИ-заключение опирается только на расчёт модели и не заменяет due diligence.
- Rate-limiter одноинстансовый (in-memory); при масштабировании заменить на Redis.
